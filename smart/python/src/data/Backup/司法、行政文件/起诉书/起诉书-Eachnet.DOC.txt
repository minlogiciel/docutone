诉讼请求：
一）判定被告立即停止侵权行为，并在被告拥有的网易网站和全国性媒体上向原告公开道歉；
二）判令被告赔偿原告经济损失人民币______元；

事实及理由：
原告方经营的易趣网拥有众多会员，是目前中国最大的网上竞标买卖平台[请确认或加入其他关于易趣网地位的说明]。易趣网所有网页，包括有关网上拍卖的网页以及支持网上拍卖的计算机软件均由原告方设计、制作。原告方依法享有这些网页和计算机软件的著作权。
2002 年 7 月 9 日[请确认日期]，被告方在其拥有的网易网站上开通了拍卖频道。原告方发现，网易网站拍卖频道的网页在整体版式、色彩、图案、栏目设置、栏目标题、方案和菜单运用方面与易趣网的拍卖网页几乎完全一样，大规模抄袭了易趣网的拍卖网页[请从专业角度审查描述是否正确]。（见证据一）
发现上述情况后，原告方对网易网站的拍卖频道进行了认真研究，发现其不但抄袭了易趣网的网页设计，而且其使用的支持网上拍卖的计算机软件也与易趣网拍卖平台支持软件相同。实际上盗用了原告方为支持易趣网拍卖平台而设计、制作的计算机软件[请从专业角度审查陈述是否正确]。（见证据二）[证据二应该是对两家网站使用的软件的 Java Script 是否相同以及相应依据的说明，最好以专家证言的形式提出。请贵公司的技术人员草拟此技术说明，然后由我们修改成符合诉讼要求的专家证言，最后交权威专家签字]鉴于原告方从未许可过被告方使用原告方为易趣网的拍卖平台涉及的网页和计算机软件，被告方抄袭易趣网拍卖网页和盗用易趣网拍卖平台支持软件的行为已经构成对原告方合法著作权的侵犯。
此外，由于原告方曾经与被告方合作，在被告方拥有的网易网站上设置了指向易趣网拍卖平台的链接，很多易趣网拍卖平台的用户经常通过网易网登陆易趣网的拍卖网页。被告方利用这一点，在合作终止后立即推出几乎完全模仿易趣网拍卖平台的拍卖频道。不仅直接抄袭易趣网的网页，还照抄了易趣网的拍卖操作模式，使得大量易趣网拍卖平台的用户错误地将网易拍卖频道误认为易趣网拍平台，不再访问易趣网的拍卖网页。有的用户甚至直接在网易拍卖频道使用自己在易趣网获得的信用积分，在发现不能使用积分时向易趣网提出投诉。严重影响了易趣网的商业信誉。这种局面造成易趣网拍卖网页的访问量明显下降，大量用户被网易网截流，而被告方却从中获利（见证据三）[证据三是有关易趣网访问量下降、客户流失和因此遭受损失的证据，请贵公司财务人员核算贵公司的损失数额。如无准确数额，可以提出估算损失额并说明估算的方法。比如，可以采用2002 年 7 月前三个月平均访问量、利润和 7 月以后访问量、利润对比的方式]。被告方为了夺取易趣网易趣网的客户资源，获取网上拍卖服务的市场份额，不惜采取抄袭易趣网的网页和剽窃拍卖操作模式的手段。这种误导消费者，混淆服务提供商的行为已经违反了《中华人民共和国反不正当竞争法》的相关规定，严重影响了原告方的合法经营和商业信誉，构成不正当竞争。
综上所述，被告方抄袭易趣网网页设计和盗用易趣网拍卖平台支持软件的行为侵犯了原告方的著作权，同时构成对原告的不正当竞争行为，侵犯了原告方的合法权益。故诉至贵院，请求判如所请。
