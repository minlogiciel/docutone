诉讼请求：
一）判定被告立即停止侵权行为，并在被告拥有的网易网站和两份以上全国性媒体上向原告公开道歉；
二）判令被告赔偿原告经济损失人民币 200 万元；
 三) 判令被告承担本案的全部诉讼费用。
事实及理由：
原告方经营的易趣网拥有众多会员，是目前中国最大的网上竞标买卖平台。易趣网所有网页，包括有关网上拍卖的网页以及用于支持各网页效果的采用 Javascript 编写的源文件均由原告方设计、制作。
原告方依法享有这些网页和计算机软件的著作权。
2002 年 7 月 8 日，被告方在其拥有的网易网站上开通了拍卖频道。原告方发现，网易网站拍卖频道的网页在整体版式、色彩、图案、栏目设置、栏目标题、检索项目等方面与易趣网的拍卖网页几乎完全一样，大规模抄袭了易趣网的拍卖网页。（具体内容的比较及相关证据参见附件一）
发现上述情况后，原告方对网易网站的拍卖频道进行了认真研究，发现其不但抄袭了易趣网的网页设计，而且为了实现各网页效果，其采用的 Javascript 源文件与原告方所编写的 Javascript 源文件极其相似，由于 Javascript 源文件可以直接从网页上获得，被告方实际上是直接获取了原告方的 Javascript 源文件并加以使用。（关于Javascript 源文件的比较参见附件二）。
鉴于原告方从未许可过被告方使用原告方易趣网拍卖平台涉及的网页和计算机软件，被告方抄袭易趣网拍卖网页和盗用易趣网Javascript 源文件的行为已经构成对原告方合法著作权的侵犯。

此外，由于原告方曾经与被告方合作，在被告方拥有的网易网站上设置了指向易趣网拍卖平台的链接，很多易趣网拍卖平台的用户曾经通过网易网登陆易趣网的拍卖网页。被告方利用这一点，在合作终止后立即推出几乎完全模仿易趣网拍卖平台的拍卖频道。不仅直接抄袭易趣网的网页，还照抄了易趣网的拍卖操作模式，使得大量易趣网拍卖平台的用户错误地将网易拍卖频道误认为易趣网拍平台，不再访问易趣网的拍卖网页。有的用户甚至直接在网易拍卖频道使用自己在易趣网获得的信用积分，在发现不能使用积分时向易趣网提出投诉。严重影响了易趣网的商业信誉。这种局面造成易趣网拍卖网页的访问量明显下降，大量用户被网易网截流，而被告方却从中获利。被告方为了夺取易趣网易趣网的客户资源，获取网上拍卖服务的市场份额，不惜采取抄袭易趣网的网页和剽窃拍卖操作模式的手段。
这种误导消费者，混淆服务提供商的行为已经违反了《中华人民共和国反不正当竞争法》的相关规定，严重影响了原告方的合法经营和商业信誉，构成不正当竞争。
综上所述，被告方抄袭易趣网网页设计和盗用易趣网拍卖平台支持软件的行为侵犯了原告方的著作权，同时构成对原告的不正当竞争行为，侵犯了原告方的合法权益。故诉至贵院，请求判如所请。
