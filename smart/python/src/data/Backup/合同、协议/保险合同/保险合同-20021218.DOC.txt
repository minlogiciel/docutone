根据《中华人民共和国合同法》和《中华人民共和国保险法》有关规定，甲乙双方在平等互利的基

础上，经友好协商，就甲方向乙方投保 太平团体终身重大疾病保险（分红型）事宜，达成如下协议。

1、 投保范围：

年龄在 16 周岁至 65 周岁，身体健康，能正常工作或劳动的在职员工。

2、 保险责任：

2.1重大疾病保险金

自合同生效日或最后复效日起一百八十日内，被保险人经本公司指定或认可的医疗机构确诊

初次罹患本合同所指重大疾病，本公司无息返还所交保险费，同时对该被保险人的保险责任终止。

自合同生效日或最后复效日起一百八十日后（以较迟者为准），被保险人经本公司指定或认可的

医疗机构确诊初次罹患本合同所指重大疾病，本公司按本合同约定的保险金额给付重大疾病保险

金，同时对该被保险人的保险责任终止。

2.2 身故保险金

被保险人身故，本公司按本合同约定的保险金额给付身故保险金，同时对该被保险人的保险

责任终止。

3、 除外责任：

3.1 投保人或被保险人的故意隐瞒、欺诈行为；

3.2投保人、受益人对被保险人的故意伤害；

3.3被保险人犯罪、企图犯罪、拒捕；

    3.4被保险人殴斗、醉酒、故意自伤及服用、吸食、注射毒品；

3.5被保险人在本合同生效日起两年内或最后复效日起两年内（以较迟者为准）自杀；

3.6被保险人在合同生效日或最后复效日前（以较迟者为准），曾患有、或获告知患有、或接受治疗

的本合同所指重大疾病；

3

3.7职业病、先天性疾病、先天性畸形、变形或染色体异常；

3.8险人酒后驾驶、无有效驾驶执照驾驶或者驾驶无有效行驶证的机动交通工具；

3.9险人未遵医嘱，私自服用、涂用、注射药物；

3.10保险人从事潜水、跳伞、攀岩运动、探险活动、武术比赛、摔跤比赛、特技表演、赛马、赛车等

高风险运动；

3.11保险人患有艾滋病或者感染艾滋病毒（HIV 阳性）期间；

3.12 保险人从事或参与恐怖主义活动、邪教组织活动；

3.13战争、军事行动、暴乱或者武装叛乱；

3.14核爆炸、核辐射或者核污染。

发生上述情形而导致被保险人身故的，本合同对该被保险人的保险责任终止。投保人已交足两年以

上保险费的，本公司退还本合同现金价值；投保人未交足两年保险费的，本公司在扣除手续费后，退

还保险费。

未尽事宜以《太平团体终身重大疾病保险（分红型）》条款为准。

6、 保险金的申请：

（1） 重大疾病保险金的申请

由被保险人作为申请人，填写保险金给付申请书，并凭下列证明、资料向本公司申请给付保险金：

6.1．投保人证明、保险单或保险凭证；

6.2．最近一期交费凭证；

6.3．被保险人的户籍证明与身份证件；

6.4．本公司指定或认可的医疗机构出具的被保险人病历、病理、血液及其他科学方法检验报告等诊

断证明文件；

6.5．本公司需要的其它有关文件和资料。

（2） 身故保险金的申请

由受益人作为申请人填写保险金给付申请书，并提交下列证明文件和资料向本公司申请给付保险金：

1．投保人证明及保险单或其它保险凭证；

2．最近一期交费凭证；

3．受益人的户籍证明或身份证明；

4．法律上认可的机构出具的被保险人死亡证明；

5．如被保险人为宣告死亡，申请人需提供人民法院出具的宣告死亡证明文件；

6．被保险人的户籍注销证明；

7．本公司认为必要的与确认保险事故的性质、原因等有关的其它证明和资料。

本公司在收到申请人的保险金给付申请书及上述证明、资料后，对确定属于保险责任的，在与申请

人达成有关给付保险金数额的协议后十日内，履行给付保险金的义务；对不属于保险责任的，向申请

人发出拒绝给付保险金通知书。如被保险人在宣告死亡后又生还，身故保险金受益人应在知道或应当知

道被保险人生还后三十日内，向本公司无息退还已领取的保险金。被保险人对本公司请求给付保险金的

权利，自其知道或应当知道保险事故发生之日起五年不行使而消灭。

7、 特别约定：

7.1、乙方为甲方的每一名被保险人出具一张分单；

      2、甲方对符合甲方内部政策的员工给予分单，并以书面形式通知乙方。该员工拿到分单之后在理

赔时与乙方的任何争议与甲方无关；但若该员工办理退保时则需向乙方出具甲方开出的员工

离职证明及退保申请书，乙方按条款相关规定为离职人员办理退保手续；

      3、甲方可以增减保险单中包含的任意具体被保险人；（甲方因在职人员变动需要增加被保险人的

应以书面形式或乙方认可的其它形式通知乙方，经乙方审核同意，于收取保险费的次日零时

起开始承担保险责任；甲方因被保险人离职或其它原因需要减少被保险人的，应 以书面形式

通知乙方，本合同对该被保险人所承担的保险责任自通知到达之日 二十四时起终止，投保人

已交足两年以上保险费的，本公司退还该被保险人的现金价值；投保人未交足两年保险费的，

本公司在扣除手续费后，退还该被保险人的保险费。如投保人要求的退保日期在通知到达日之

后，则该被保险人资格自该退保日零时起丧失）。
